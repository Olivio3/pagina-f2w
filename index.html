SOBRE A FORMAÇÃO

Uma jornada completa no universo WebGIS e WebApps

A Formação 2W foi desenhada para profissionais e estudantes que querem dominar o desenvolvimento de aplicações geoespaciais modernas — desde a configuração do ambiente até o deploy em produção, passando por bancos de dados espaciais, APIs, containers e visualização interativa.

Python PostgreSQL / PostGIS GeoServer Docker Leaflet.js OpenLayers Flask / Django Plotly / Dash Google Earth Engine GitHub Pages HTML / CSS / JS Streamlit
